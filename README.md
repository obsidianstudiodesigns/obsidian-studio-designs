# Obsidian Studio Designs

Dark & gold 3D interactive website for Obsidian Studio Designs — website and app development.

**Live site:** https://obsidianstudiodesigns.github.io/obsidian-studio-designs/

## Features

- Full-screen video landing page
- Clickable 3D devices built in pure CSS — laptop boots HTML code, phone rains binary, then each scrolls to its section
- Scroll-driven 3D rotations, letter-by-letter title reveals, gold particle field
- Unique animated 3D object for every service
- Contact form (via FormSubmit) delivering to obsidianstudiodesigns@gmail.com
- Floating WhatsApp chat button
- Fully responsive with mobile hamburger navigation
